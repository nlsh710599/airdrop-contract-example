# airdrop-contract-example

### Requirement:

Add a file in root folder.

1. Create .private-keys and input deployer's private key
2. Rename .env.template to .env file and fill up params

### Scripts:

deploy:

```bash
$ yarn deploy-prod
```

test:

run ganache

```
$ ganahe
```

perform test

```bash
$ truffle test
```
